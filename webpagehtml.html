Some Things I've Learned About HTML

HTML is the standard language used to create webpages. It's made up of various elements and tags that define the structure and content of a webpage. One important thing I've learned is that HTML documents are made up of different types of elements:

  • The <html> tag: defines the start and end of an HTML file
  • The <head> tag: contains data meant for the beginning of the file like the title
  • The <body> tag: contains the main content of the page

Another important concept is using different types of elements for structuring content:

  • The <h1> to <h6> tags for headings, with <h1> being the largest and <h6> the smallest
  • The <p> tag for paragraphs, allowing you to have text in blocks
  • The <a> tag for creating links

Some Things I've Learned About CSS

CSS is used to control the layout and appearance of HTML elements. Here are a few key things I've learned about CSS:

  • CSS is used to apply styles like colors, fonts, margins, or padding to elements in an HTML file.
  • CSS selectors target HTML elements, and you can apply styles to specific elements using classes (.class) and IDs (#id).

Some Things I've Learned About JavaScript

JavaScript is used to add functionality and interactivity to a webpage. Here are a few key things I've learned:

  • Updating Date & Time: JavaScript can dynamically update elements, like displaying the current date and time. 
    setInterval(updateDateTime, 1000);
  • Scrolling to Sections: JavaScript allows smooth scrolling to different sections when a button is clicked. 
    document.getElementById("sectionId").scrollIntoView({ behavior: "smooth" });
  • Scroll-to-Top Button: Using JavaScript, we can detect when the user scrolls down and show a button to return to the top. 
    window.addEventListener("scroll", function () { ... });
